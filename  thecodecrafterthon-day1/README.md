# CLI Calculator

I built a simple command-line interface (CLI) tool for performing basic arithmetic operations directly on the terminal.

it can do the following:
* Addition, Subtraction, Multiplication, and Division.
* It can also handle errors for "division by zero" and invalid inputs.
* It is easy to follow prompts for user input.

HOW TO RUN IT

Make sure you have GO downloaded in your vs code or the machine you want to use.

 1. clone the repository:  git clone https://github.com/abrahamzion01/the-codecrafters-.git.
 
 2. cd cli-calculator

HOW TO RUN
go run cli.go

example of how to use it:
Enter first number: 10
Enter second number: 5
Enter operator (+, -, *, /): *

Result: 50
